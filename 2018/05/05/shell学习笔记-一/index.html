shell学习笔记-一

发表于 2018-05-05 | 分类于 shell | 阅读次数:

字符串

字符串是shell编程中最常用最有用的数据类型（除了数字和字符串，也没啥其它类型好用了），字符串可以用单引号，也可以用双引号，也可以不用引号

单引号

单引号特点

  • 单引号里的任何字符都会原样输出，单引号字符串中的变量是无效的；
  • 单引号字串中不能出现单引号（对单引号使用转义符后也不行）。

双引号

双引号的特点

  • 双引号里可以有变量
  • 双引号里可以出现转义字符

字符串长度

使用 # 可以获取字符串长度

截取字符串

格式: string:{start}:{end},位置从0开始, 如果start为负数,则从0开始截取,如果end大于字符串的长度,则截取字符串最大值；也可以使用expr substr start end，但expr的start从1开始。

ps: 当要使用shell session命令时可以用$()或者 ` ` 把命令包裹

定位字符位置

使用 expr index 命令可以定位字符，位置是从1开始计算

ps: 如果参数为字符串，则只识别首字符。

字符串运算符

下表列出了常用的字符串运算符，假定变量 a 为 “abc”，变量 b 为 “efg”：

运算符 说明 举例
= 检测两个字符串是否相等，相等返回 true。 [ $a = $b ] 返回 false。
!= 非等于 [ $a != $b ] 返回 true。
-z 检测字符串长度是否为0，为0返回 true。 [ -z $a ] 返回 false。
-n 检测字符串长度是否为0，不为0返回 true。 [ -n $a ] 返回 true。
str 检测字符串是否为空，不为空返回 true。 [ $a ] 返回 true。
