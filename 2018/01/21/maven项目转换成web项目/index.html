maven项目转换成web项目

发表于 2018-01-21 | 分类于 maven

用maven创建一个webapp的项目时，在eclipse用tomcat容器识别不了，那是因为用maven创建出来的项目不是一个web项目。

解决办法

  1. 首先创建一个maven的webapp项目(不要点create a simple project，否则到不了这个页面)


  2. 右键工程属性选择”Project Facets” 点击 “Convert faceted form”


  3. 勾选Dynamic Web Module


  4. 点击Further configuration available…，更改web目录为 “src/main/webapp”


  5. 再右键工程属性选择”Deployment Assembly”,把webapp部署到tomcat里,并把依赖添加到WEB-INF/lib下


  6. 如果想要更改Context root path，可以右键工程属性选择”Web Project Settings”进行更改

  7. 最后用tomcat进行add and remove时就能发现web工程了。
