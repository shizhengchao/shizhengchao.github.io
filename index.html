Centos安装Node.js

发表于 2017-06-20 | 分类于 nodejs
  1. 下载源码
    cd /usr/local/src/
    wget [https://nodejs.org/dist/v8.10.0/node-v8.10.0.tar.gz]

  2. 解压源码
    tar zxvf node-v0.10.24.tar.gz

  3. 进入到node根目录,编译并安装:
    cd node-v8.10.0
    ./configure –prefix=/usr/local/node/8.10.0
    make && make install

linux常用命令记录

发表于 2017-03-20 | 分类于 笔记

基本命令

  • df -h : 查看磁盘使用情况
  • du -h –max-depth=1 : 查看当前目录下的所有目录的使用情况 参数 –max-depth=1代表目录深度
  • find . -type f -size +800M -print : 找出当前目录下及子目录下的大于800M的文件
  • rm -rf xxx:-f强制删除，-r如果是目录，删除包含子目录的所有文件
  • mkdir folder:创建文件夹
  • cp abc.log /dir/:复制abc.log文件到/dir目录下
  • mv abc.log /dir:移动或者重命名文件
